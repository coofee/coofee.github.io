Hugo

1.1 发布到本地预览网站


$ hugo server --theme=greyshade --buildDrafts --watch

1.2 发布网站到github


$ hugo  --theme=greyshade --buildDrafts
$ cd public
$ git add * 
$ git commit -a -m 'add all'
$ git push origin master
